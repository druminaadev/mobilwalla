# Pattern Icons

Place your background pattern icons here:

## Required Files:
- `scissors.png` - Salon scissors icon (80x80px)
- `dryer.png` - Hair dryer icon (80x80px)
- `comb.png` - Hair comb icon (80x80px)

## Specifications:
- Format: PNG with transparent background
- Size: 80x80px
- Style: Simple line icons
- Color: Black or dark (will be rendered at 5% opacity)

These icons are used as subtle background decorations on the Welcome screen.
